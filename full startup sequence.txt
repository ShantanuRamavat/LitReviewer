Here's the full startup sequence after a reboot:

1. Start Docker infrastructure (PostgreSQL, Redis, Qdrant)


cd "d:\Study\Projects\Multi-Agent Research Team\infrastructure"
docker compose up -d postgres qdrant redis
2. Start the backend (in a separate terminal)


cd "d:\Study\Projects\Multi-Agent Research Team\backend"
uv run uvicorn app.main:app --reload
Wait for: All startup tasks complete. Nexus Research API is accepting requests.
(First run after reboot takes ~1 min to load the embedding model from cache)

3. Start the frontend (in another separate terminal)


cd "d:\Study\Projects\Multi-Agent Research Team\frontend"
npm run dev
4. Open http://localhost:3000

Prerequisites that must be installed: Docker Desktop (must be running), Python with uv, Node.js

To stop everything:


# Stop Docker services
docker compose -C "d:\Study\Projects\Multi-Agent Research Team\infrastructure" down
# Press Ctrl+C in the backend and frontend terminals
